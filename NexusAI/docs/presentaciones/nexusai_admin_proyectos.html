Decisiones técnicas

Lo que decidimos por el camino

🗄️

Una sola base de datos

PostgreSQL + pgvector en lugar de una base vectorial aparte. Backup estándar, menos infra.

PostgreSQL 16pgvector
🔄

IA intercambiable

Cambiar de Gemini a GPT-4o-mini es solo modificar variables de entorno. Sin lock-in.

Gemini (dev)GPT-4o-mini (prod)
🔐

Seguridad en 3 capas

Bearer key + firma HMAC-SHA256 + anti-replay en Redis. La API key nunca llega al navegador.

HMAC-SHA256Redis nonces
🧩

Plugin no invasivo

Tipo local: se instala en Moodle sin tocar el core. Compatible 4.1–4.5 LTS.

local_nexusaiMoodle 4.1–4.5
